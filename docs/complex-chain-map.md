# 复杂链关系图

这张图用于帮助理解：长期使用后，系统承接的复杂度主要来自多条链并行、策略版本演化、执行事件、偏离事件与复盘回流之间的关联，而不只是单条聊天记录越来越长。

```mermaid
flowchart LR

    M["市场背景<br/>宏观 / 新闻流 / 快讯 / 社媒 / 情绪环境"]
  end

  subgraph SA["策略资产演化"]
    V1["策略版本 V1<br/>趋势跟随 / 宽止损"]

    SR["signal_rule"]
    RR["risk_rule"]

    V1 --> V2
    V2 --> SR
    V2 --> RR
    V2 --> CR
  end

  subgraph BTC1["BTC 波段多链（展开）"]
```
